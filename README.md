**Currency Converter Web App**
**Overview**
A simple and responsive Currency Converter web application that allows users to convert one currency into another using real-time exchange rates.
This project focuses on API integration, async JavaScript, and clean UI logic.
**Features**
Convert between multiple international currencies
Real-time exchange rates
Clean and minimal UI
Error handling for invalid inputs
Responsive design
**Technologies Used**
HTML
CSS
JavaScript (ES6)
Exchange Rate API (used for live currency data)
**What I Learned**
Working with REST APIs
async / await and fetch()
DOM manipulation
Handling user input and edge cases
